Stocks and SAT

In this section, we cover the uses of the Lag, Difference, and Sign data transformations in the creation of a time series of Google stock from April 2014 to April 2015. The time series will color coded to display overall increase and decrease of stock price by month.

We can start with a basic time series to better understand what we are looking at:


ggplot() + geom_line(data = stock, aes(Date, Adj.Close), colour = "navy") + 
    scale_y_continuous(limits = c(475, 600)) + theme(panel.grid.minor = element_blank(), 
    panel.grid.major = element_line(colour = "gray"), panel.background = element_rect(fill = "white"), 
    plot.title = element_text(size = rel(1.5))) + labs(title = "Google Stock Price April 2014 to April 2015", 
    y = "Stock Price")

Next, we endeavor to enhance this detailed time series with information about when the stock is increasing or decreasing overall.

The Difference Transformation and Lag Transformation


The difference function
diff(x1, x2,…, xn),
x1-xn vectors
is relatively self explanatory. We use it to subtract columns of data from one another.

We use the lag function
lag(x), x a vector
to shift a time series down a column by one unit, replacing the first entry with 'NA'. This is useful for finding the day to day (or month to month, year to year, etc.) difference between two values. In this case, we use it to shift the column with the stock prices of Google on the last date of each month down an index. We then use the difference function to subtract the lagged column from the original column.

# shift lag column of stock DF down an index and put an NA in the first
# entry.
stock$lag <- c(NA, head(stock["lastdayprice"], dim(stock)[1] - 1)[[1]])

# subtract the lag column from the original price column to find the month
# to month difference
stock$difference <- stock$lastdayprice - stock$lag

# manually calculate the last difference given knowledge of stock value
stock$difference[252] <- 542.56 - stock$lag[252]

# fill in 0s with difference values
temp = 1
for (i in 2:251) {
    if (!(stock$difference[i] == stock$difference[i + 1])) {
        stock$difference[temp:i] <- stock$difference[i + 1]
        temp = i
    }
}

for (i in 2:251) {
    if (stock$difference[i] == 0) {
        stock$difference[i] <- stock$difference[i - 1]
    }
}


We are now capable of plotting the month to month change of Google stock:

ggplot() + geom_line(data = stock, aes(Date, difference), colour = "navy") + 
    theme(panel.grid.minor = element_blank(), panel.grid.major = element_line(colour = "gray"), 
        panel.background = element_rect(fill = "white"), plot.title = element_text(size = rel(1.5))) + 
    labs(title = "Google Stock Price Differences April 2014 to April 2015", 
        x = "Difference", y = "Stock Price")

The Sign Transformation


We use the sign function
sign(x), x a vector
to simply determine whether a number is positive, negative, or 0 by mapping it to 1, -1, or 0 respectively. In this case, we look at the sign of the month to month difference in the stock price.

# Assign a 1,-1, or 0 to positive, negative, and 0 numbers respectively in
# the difference column.
stock$sign <- sign(stock$difference)

stock$pos <- stock$Adj.Close
stock$neg <- stock$Adj.Close


We are now capable of plotting the sign of the month to month change of Google stock:

ggplot() + geom_line(data = stock, aes(Date, sign), colour = "navy") + theme(panel.grid.minor = element_blank(), 
    panel.grid.major = element_line(colour = "gray"), panel.background = element_rect(fill = "white"), 
    plot.title = element_text(size = rel(1.5))) + labs(title = "Google Stock Price Differences April 2014 to April 2015", 
    x = "Sign of Price Change", y = "Stock Price")

Here is the final product of our use of the lag, difference, and sign functions:


Where the line is green, Google saw an overall increase in price for that month. Similarly, where the stock is red, Google saw an overall decrease for that month. With a visual like this, we can search for trends in the day to day behavior of a stock during times when it sees an overall increase or decrease.

for (i in 1:252) {
    if (stock$sign[i] < 0) {
        stock$pos[i] <- 0
    } else {
        stock$neg[i] <- 0
    }
}

ggplot() + geom_line(data = stock, aes(Date, pos), colour = "green") + geom_line(data = stock, 
    aes(Date, neg), colour = "red") + scale_y_continuous(limits = c(475, 600)) + 
    theme(panel.grid.minor = element_blank(), panel.grid.major = element_line(colour = "gray"), 
        panel.background = element_rect(fill = "black"), plot.title = element_text(size = rel(1.5))) + 
    labs(title = "Google Stock Price", y = "Stock Price")
